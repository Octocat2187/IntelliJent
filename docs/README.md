# IntelliJent

## Installation

To install and run the project:

1. **Backend**: Navigate to the `backend` folder and run:
   ```
   ./gradlew run
   ```

2. **Frontend**: Navigate to the `frontend` folder and run:
   ```
   npm run dev
   ```

## Features Not Included

The following features are not currently included in this project:

- User and Professor Profiles
- Course Prerequisite Information
